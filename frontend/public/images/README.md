# Folder: public/images

Folder ini digunakan untuk menyimpan aset gambar utama (foto atau ilustrasi berukuran besar) dari Figma, seperti:
* **Foto Hadiah / Pedestal Hadiah** (contoh: sepeda, TV, kulkas)
* **Karakter Maskot atau Dekorasi Utama**
* **Voucher / Kupon Hadiah**

Format yang disarankan: `.png` (transparan), `.webp` (ringan), atau `.jpg`.
