# Folder: public/svg

Folder ini digunakan untuk menyimpan elemen-elemen dekoratif grafis berbasis vektor (.svg) dari Figma, seperti:
* **Bentuk Awan Kustom (Custom Clouds)**
* **Bentuk Bintang Dekorasi (Decorative Stars/Sparkles)**
* **Gelombang / Ombak Bukit Rumput di bagian bawah**
* **Background shapes lainnya**

Format yang disarankan: `.svg`.
